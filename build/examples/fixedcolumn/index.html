Fixed Column

Fixed Column is a layout where each column has a fixed width and horizontally centered in the screen. See CQ5-30653. It is achieved by the following selector structure:

/**
 * Indicates the container of the columns.
 * @see https://issues.adobe.com/browse/CQ5-30653
 */
.coral-FixedColumn

/**
 * Indicates the column.
 */
.coral-FixedColumn-column

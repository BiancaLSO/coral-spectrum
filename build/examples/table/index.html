Basic example

The basic styling of a table includes horizontal dividers.

Hover rows

Bordered table

Add .coral-Table--bordered to add a border to the table and between columns.

Bordered table with hover

The modifiers can be combined to produce a new style. In this case we added .coral-Table--bordered and .coral-Table--hover to create the desired effect.

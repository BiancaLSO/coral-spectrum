BlackBar

Current Version: 3.0.5-pre
Related Components: ShellWidget, Page
Github: coralui-component-shell

The BlackBar component is the part of the shell that goes over the main page. It usually holds the current page's title, a hamburger icon for toggling the outer rail, and a back button. Most of these items can be toggled off, and additional items can be added to the right hand side of the component.

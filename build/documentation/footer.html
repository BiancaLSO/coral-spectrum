Footer

Current Version: 3.0.6-pre
Related Components: ShellWidget, Page
Github: coralui-component-shell

The footer gets displayed at the bottom of a shell's main page area. Its styling is such that it will stick to the bottom of the window when the page content fits within the available vertical space, but will trail the page content when it doesn't fit.

The widget shows a default copyright text (that can be reset) and allows for adding a number of footer links. Links can be preset via the DOM (see example) and be added and removed via JavaScript (see addItem() and removeItem()).

When Footer is used outside the context of CUI.Page, it will not display correctly, hence the examples below are set to not show at all. This is how the footer would appear inside the context of CUI.Page:

Wait Indicator

Current Version: 2.3.8-pre

Related Components:

Use a wait indicator to show a process is in progress that will occur for an indefinite amount of time by applying coral-Wait to an element.

Sizing

Adding the coral-Wait--large class will result in a larger wait indicator.

Centering

Adding the coral-Wait--center CSS class will make the wait to be centered in the parent element. The parent must be positioned.

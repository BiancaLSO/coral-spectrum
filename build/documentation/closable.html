Closable

Current Version: 3.0.5-pre
Related Components: ShellWidget
Github: coralui-component-shell

The Closable widget is a base class for shell widgets that can be in a closed state. It is not intended to be used directly and cannot be instantiated using data-init. Widgets that derive from CUI.Closable include CUI.Brand, CUI.BreadcrumbBar and CUI.OuterRail.

Markup Example

Notice that the first element has a data-is-closed attribute set. Since CUI.Brand derives from CUI.Closable, this results in the component to enter the closed state.

Progress Bar

Current Version: 2.3.3-pre

Related Components:

Use the coral-Progress container with a coral-Progress-bar to create a progress bar. The coral-Progress-status is used to indicate the filled portion of the progress bar.

A <label> tag with the class coral-Progress-label can be included immediately after a progress barto display progress as a numeric value.

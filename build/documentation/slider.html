Labeled Sliders

              

              
The labeled slider uses the same options/markup as the slider, but allows you to specify a list of labels for the slider's ticks.  Note that it is implemented in a subclass, specified via the labeled-slider data initializer. Labeled sliders are not currently supported in a vertical orientation.

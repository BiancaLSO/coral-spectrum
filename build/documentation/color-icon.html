Color Icons

Current Version: 1.0.3
Relevant Specifications: Color Icons
Related Components: Icon
Github: coralui-component-coloricon

Color icons are not included in the core of CoralUI. To enable the use of the color icon selectors, the coralui-component-coloricon component must be included as a dependency in your CoralUI build.

Color Icon Selectors

The color icons require the coral-ColorIcon base selector, but otherwise work in a similar way to the monochrome icons. Once included, you can use coral-ColorIcon and the desired variant and size selectors in the same way you would use the monochrome icon selectors.

Color Sizes

Similar to the monochromo icons, there are 4 different color icon sizes:

                              coral-ColorIcon--sizeXS
                              coral-ColorIcon--sizeS
                              coral-ColorIcon--sizeM
                              coral-ColorIcon--sizeL

You can also mix and match and load html and json asynchronously depending on your preference. This could be very useful for a solution like Marketing Cloud that will load in multiple other solutions data where each solution may follow a different implementation.

                      
NavigationView using all server-generated markup

                  
It is also possible to load in everything from the server without doing anything asynchronously. This is the solution that AEM has implemented. An example is not provided here because it should be self-explanatory. Be aware that if there are markup changes your implementation's markup will also need to change. If you use the JSON implementation then you should get changes automatically unless the underlying JSON structure has changed.

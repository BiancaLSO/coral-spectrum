Quicktip

              

              

                Quicktip is a special mode of Tooltip that provides a quick way to define a
                Tooltip on any element. The Tooltip will automatically be created and shown when
                the user mouses over the element. Quicktip mode also works for elements that have been dynamically inserted into the
                document.

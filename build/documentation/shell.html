Shell

Current Version: 3.0.6-pre
Related Components: ShellWidget
Github: coralui-component-shell

The shell widget orchestrates the shell application's breadcrumb bar and the remainder of the page. It may also propagate brand settings to both the breadcrumb bar, and the outer rail's brand section.

The shell widget is expected to be attached to the body tag. Because of this, iframes have been used in the examples below.

CoralUI Shell

Controllers

The CoralUI shell is a set of components, and two controllers. The controllers (CUI.Shell and CUI.Page, denoted in the sample below by the green dotted boxes; CUI.Page being the inner box) control the interaction between the components. Drill into the 'Controllers' section on the right hand side for more information on each of the controllers.

Components

Mouse over a name in the list of components below to see where it lives within a typical shell page. Drill into the 'Components' section on the right hand side menu to find more information on each of the components.

Services

Shell defines a number of services that shell components rely on. Find out more about each service by selecting Services from the menu.

Internal

Shell defines some classes that are used by other shell components. They are not intended to be used by end users directly.

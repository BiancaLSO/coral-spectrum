HelpPopover JavaScript

The HelpPopover is one of many components that are specific to Digital Marketing. It can be instantiated using a data-init flag or by invoking JavaScript directly. It will most likely be used in conjunction with the BlackBar component but can be used outside of here if desired. Please refer to the example below to see this used within the BlackBar. You can provide as much or as little markup as you desire. If you don't come with any markup it will be generated. The example below displays the minimum amount of code that you need to add in order to initialize the HelpPopover.

Class Description: This widget controls the HelpPopover component

This class will most likely be used in conjunction with the BlackBar component but can be initialized
elsewhere if desired. Every product will want to make sure that they pass in the search product so that
they get results that are specific to their specific product. It will default to pulling in help topics
from the marketing cloud.

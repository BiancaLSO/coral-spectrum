Minimal Button

Current Version: 2.3.8-pre

Related Components: Button,Icon

The coral-MinimalButton class creates a button element without any other styling. It's intended use is to wrap an icon which has the coral-MinimalButton-icon class. This creates an icon-only button without any additional size, position, or styles.

Note that MinimalButton will scale if a size is added via CSS. See examples below.

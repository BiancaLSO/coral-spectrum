MinimalButton

Current Version: 2.3.11-pre
Related Components: button, icon
Github: coralui-core

MinimalButton a button element without any styling. It's typically intended to wrap an icon which has the coral-MinimalButton-icon class. This creates an icon-only button without any additional size, position, or styles. Note that MinimalButton will scale if a size is added via CSS. See examples below.

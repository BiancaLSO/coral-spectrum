Checkbox

Current Version: 2.2.13-pre

Related Components:

Checkboxes are styled using coral-Checkbox on the input element, with enclosed span tags styled as coral-Checkbox-checkmark and coral-Checkbox-description.

Checkbox include checked and unchecked states with pure CSS. They can support an indeterminate state using javascript.

Examples of the required markup are shown below.

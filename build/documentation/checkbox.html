Checkbox

Current Version: 2.4.23
Github: coralui-core

Checkbox functions like a standard browser checkbox. It is typically used to allow a user to select multiple options from a list. Checkbox includes checked and unchecked states using pure CSS and can support an indeterminate state using JavaScript.

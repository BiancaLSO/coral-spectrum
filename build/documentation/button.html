Button

Current Version: 2.3.8-pre

Related Components: MinimalButton,CycleButtons

A CoralUI Button is designated using coral-Button. This can be specified for tags other than <button>.

By default a CoralUI Button is gray. The available type selectors are:

.coral-Button--primary
.coral-Button--secondary
.coral-Button--quiet
.coral Button--warning

CoralUI styles also support the disabled attribute.

Square Buttons / Icon Buttons

                
An square style of button is available.  Use coral-Button--square to achieve this.

                  set.  Icons selectors should be placed on a <i> tag.

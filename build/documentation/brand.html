Brand

Current Version: 3.0.5-pre
Related Components: ShellWidget, Closable, Page, OuterRail, BreadcrumbBar
Github: coralui-component-shell

Brand is a component that displays a shell application's brand usually consisting of an icon followed by a label. Brand expects its target element to be an anchor.

Endor.store

Current Version: 3.0.5-pre
Related Components: ShellWidget
Github: coralui-component-shell

CUI.Endor.store is used by CUI.ShellWidget to persist options (if so configured: option storage is opt-in, per option that's to be stored). The default implementation stores the option as a cookie, using the jquery-cookie plugin. Custom storage schemes may be plugged in by overriding the object.

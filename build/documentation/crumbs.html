Crumbs

Current Version: 3.0.5-pre
Related Components: ShellWidget, Shell, Breadcrumb Bar
Github: coralui-component-shell

The shell crumbs component is usally placed in the breadcrumb bar to visualize the client's navigational path to the current page. When there's insufficient room to show all the crumbs, the component can truncate the crumbs to make things fit.

List

Current Version: 2.3.8-pre

Related Components:

The selector for a list is coral-List. Each list item should have the class coral-List-item.

Besides regular ordered and unordered lists, which don't need a specific class, two custom styles are provided: 

.coral-List--minimal
.coral-List--condensed

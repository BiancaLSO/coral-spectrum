Table

Current Version: 2.3.8-pre

Related Components:

Tables require the main class of coral-Table.

In addition, the following subtypes are provided for rows, headings, and cells.

.coral-Table-row
.coral-Table-headerCell
.coral-Table-cell

Two decorative styles are provided as well:

.coral-Table--bordered
.coral Table--hover

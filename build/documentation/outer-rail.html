OuterRail

Current Version: 3.0.5-pre
Related Components: ShellWidget, Closable, Page
Github: coralui-component-shell

OuterRail is a container usually placed on the left-hand side of a shell page using a dark style. It typically contains a NavigationView and Brand. The examples below contain both.

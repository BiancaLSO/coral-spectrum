TODO: UPDATE DOCUMENTATION

A TODO requires setting the class coral-TODO on a container. The TODO should also contain TODO markup. The correct markup is shown below each example.

There are several types of TODO. Adding the correct selector determines what type will be displayed. If no type selector is given, the default is used.

The type selectors are:

.coral-TODO--something
.coral TODO--nothing
.coral-TODO--thing

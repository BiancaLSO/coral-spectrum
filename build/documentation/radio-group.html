Radio Group

Current Version: 2.3.8-pre

Related Components: Radio

The CoralUI Radio Group comes in three flavors: horizontal, vertical, and horizontal with labels below. Class names are placed on a container that wraps a set of radio buttons.

The relevant class names are:

coral-RadioGroup
coral-RadioGroup--vertical
coral-RadioGroup--labelsBelow

The default is horizontal, which requires only the coral-RadioGroup class name.

Also see Radio Button for details on the markup and CSS for a set of radio buttons.

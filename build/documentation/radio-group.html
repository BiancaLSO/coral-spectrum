Radio Group

Current Version: 2.3.3-pre

Related Components: Radio

The CoralUI Radio Group comes in three flavors; horizontal, vertical, and horizontal with labels below. Seletors are placed on a container that wraps a set of radio buttons.

The relevant selectors are:

.coral-RadioGroup
.coral-RadioGroup--vertical
.coral-RadioGroup--labelsBelow

The default is horizontal, which requires only the coral-RadioGroup selector.

Also see Radio Button for details on the markup and CSS for a set of radio buttons.

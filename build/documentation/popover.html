Popover

Current Version: 2.0.15
Github: coralui-component-popover

The Popover component provides a non-modal overlay of additional information related to a target element. Typically this occurs when the user clicks the target element.

Markup Examples

|

By default the popover does not include padding so you can customize what goes inside. Use .u-coral-padding to add the desired padding in the content.

Default Positioning

Popovers will be positioned so they remain visible. The default location is below the target element.

     horizontally.

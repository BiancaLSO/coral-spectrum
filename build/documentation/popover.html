Default Positioning

Popovers will be positioned so they remain visible. The default location is below the target element.

            When set to left, the popover will be anchored to the left
     side of its offset parent (in other words, it will use the
     left CSS property). When set to right, the
     popover will be anchored to the right side of its offset
     parent (in other words, it will use the right
     CSS property). When the element the popover is pointing at
     is right-aligned, it can be useful to set the value to
     right so the popover will appear to stay
     attached to the element when the user resizes the window
     horizontally.

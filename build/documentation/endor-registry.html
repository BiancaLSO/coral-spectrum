Endor.registry

Current Version: 3.0.5-pre
Related Components: ShellWidget
Github: coralui-component-shell

CUI.Endor.registry is not a component. It is a static JavaScript utility object that is used to store and retrieve instances of shell components and is typically only used by shell components internally. It can also be used by consumers to configure components once they have been generated. It is an efficient way to look up widget instances, for it doesn't involve any DOM searching (the registry keeps a hash table), nor any periodic checks (for ShellWidget based widgets register/unregister from their construct/destruct methods)

JavaScript API

              

              

                Alerts can be constructed using the Class constructor directly.
                The constructor expects a configuration object, which should include a reference
                to an element with the expected Alert markup (as in examples above).
                
              

                  
                  

                        
                          
                          

                  

              

                A reference to the element's alert instance is stored as data-alert
                
              

                  
                  

                        
                          var alert = $('#myAlert').data('alert');
                          // now you can call public functions
                          alert.hide();
                          

                  

              
Hide the Alert, change the content, then show it again...

                  
                  

                        
                          alert.hide().set({ content: 'Another error has occurred.'}).show();
                          

                  

              

                jQuery construction works as well. The constructor will initalized the element on
                which the the constructor is called.
              

                  
                  

                        
                          $('#myAlert').alert({
                            heading: 'ERROR',
                            content: 'An error has occurred.',
                            closeable: true
                          });
                          

                  

              
The markup will still obey jQuery functions as well.

                  
                  

                        
$('#myAlert').alert();

Alert

Current Version: 2.1.4

Related Components: Modal,Popover

An Alert is designated using coral-Alert on a container. The Alert should also contain an icon, a heading, a message, and an optional close button. The correct markup is shown below each example.

There are several types of Alert. Adding the correct selector determine what type will be displayed. If not type selector is given, the default type is 'info'.

The type selectors are:

.coral-Alert--info
.coral Alert--help
.coral-Alert--success
.coral-Alert--notice
.coral-Alert--error

The Alert close button is optional. Alert's JavaScript relies on a data-dismiss property of type 'alert'

The Alert icon is determined by the selector type given in the markup. The examples show the recommended icon for each type, but this in not enforced and any icon can be substituted.

JavaScript API

                

                  Alerts can be constructed using the Class constructor directly.
                  The constructor expects a configuration object, which should include a reference
                  to an element with the expected Alert markup (as in examples above).
                  
                

                    
                    

                          
                            
                            
                            

                    

                

                  A reference to the element's alert instance is stored as data-alert
                  
                

                    
                    

                          
                            var alert = $('#myAlert').data('alert');
                            // now you can call public functions
                            alert.hide();
                            

                    

                
Hide the Alert, change the content, then show it again...

                    
                    

                          
                            alert.hide().set({ content: 'Another error has occurred.'}).show();
                            
                            

                    

                

                  jQuery construction works as well. The constructor will initalized the element on
                  which the the constructor is called.
                

                    
                    

                          
                            $('#myAlert').alert({
                              heading: 'ERROR',
                              content: 'An error has occurred.',
                              closeable: true
                            });
                            

                    

                
The markup will still obey jQuery functions as well.

                    
                    

                          
$('#myAlert').alert();

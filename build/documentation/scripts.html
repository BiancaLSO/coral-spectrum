Base JSDoc

Class Description: The abstract class which contains methods that all classes will inherit. Base cannot be extended or instantiated and does not exist in the global namespace. If you create a class using new Class() or MyClass.extend(), it will come with Base' methods.

Constructor

Base is an abstract class and cannot be instantiated directly. Constructors are chained automatically, so you never need to call the constructor of an inherited class directly

Parameters:

Name Type Description
options Object Instance options. Guaranteed to be defined as at least an empty Object

CUI JSDoc

Class Description: The main CUI namespace.

Code

Change CUI options

You can change CUI options by defining CUI.options before you load CUI.js

<script type="text/javascript">
var CUI = {
  options: {
    debug: false,
    dataAPI: true
  }
};
</script>
<script src="js/CUI.js"></script>

preferable include the CUI.js at the bottom before the body closes

Properties:

Name Type Description
options Object Main options for CloudUI components.
Name Type Description
debug Boolean If true, show debug messages for all components.
dataAPI Boolean If true, add listeners for widget data APIs.
Templates Object Contains templates used by CUI widgets

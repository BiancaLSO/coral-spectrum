TextField

Current Version: 2.2.13-pre

Related Components: Decorated Textfield

Text input is handled with the CoralUI Textfield. It uses the selector coral-Textfield. This can be applied to both a input <input type='text'> or a text area <textarea>.

A Textfield can show placeholder text using the placeholder attribute. A disabled style is supported as well. Text areas should be supported by adding the coral-Textfield--multiline modifier.

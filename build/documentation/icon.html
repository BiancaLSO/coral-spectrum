Github: coralui-core

CoralUI ships with a set of monochrome icons that are delivered via convenience classes.

Standalone

Icons can be added by using both i and span elements.

Sizes

                    follows the shirt sizing system, from XS to L:
                    
                  

                      

                            
                              coral-Icon--sizeXS
                              coral-Icon--sizeS
                              coral-Icon--sizeM
                              coral-Icon--sizeL

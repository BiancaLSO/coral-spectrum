Form Layout

Current Version: 2.3.3-pre

Related Components: Textfield,Textarea,Input Group,Radio,Checkbox,Button

The base selector to specify a form is .coral-Form. In addition, CoralUI provides support for sevaral form layouts; aligned, vertical and horizontal.

The selectors are:

.coral-Form--aligned
.coral-Form--vertical
.coral-Form--horizontal

Examples using each type of layout are provided below.

Additional Selectors

Addiional selectors are available to assist with laying out more specific elements.

Fieldset and Fieldset Ledgend

.coral-Form-fieldset
.coral-Form-fieldset-legend

.fieldset. is used instead of <fieldset> to allow <legend> the required styling.

Field Labels and Fields

.coral-Form-fieldlabel
.coral-Form-field

A field label and its related field should be designated with .coral-Form-fieldlabel and .coral-Form-field

Field Info and Field Error

.coral-Form-fieldwrapper
.coral-Form-fieldinfo
.coral-Form-fielderror
.coral-Form-fieldwrapper--alignRight

Additional info or a special error message can be indicated using .coral-Form-fieldinfo or .coral-Form-fielderror. A field wrapper MUST be used to contain and poistion content when using the field info and field error classes. Alignment within the wrapper can be forced right using the .coral-Form-fieldwrapper--alignRight selector.

In conext examples for the addition selectors are provided as part of the alignment examples below.

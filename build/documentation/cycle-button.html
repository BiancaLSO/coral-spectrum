Respsonding to user interaction

                  

                    The following example shows how to respond to user interaction. Whenever
                    the CycleButton is clicked or tapped, a click event is
                    emitted for the newly active button.
                    
                  

                      
                      

                                
                                  
                                  
                                 
                      
Show Markup
                          

                  

                    Please note: Due to performance reasons and ease of
                    implementation, event handling needs to meet the following restrictions:
                    
                  

                  
    
                    
  • 
                      Custom event handlers should be registered after the component was
                      initialized.
                    
    • 
                    
  • 
                      Binding event handlers to the button elements directly will yield
                      inconsistent results. Please resort to registering delegate event
                      handlers on the CycleButton element instead.

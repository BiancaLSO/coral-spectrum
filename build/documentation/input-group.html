InputGroup

Current Version: 2.4.23
Related Components: textfield, decorated-textfield, button, datepicker, number-input
Github: coralui-core

A CoralUI InputGroup is a composite CSS pattern for Buttons and Textfields that share a border. It is the base style of several more complex components due to its versatility.

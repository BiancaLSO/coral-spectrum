What is CoralUI?

CoralUI or CUI (pronounced "cooey") is an implementation of the CloudUI visual style from Adobe XD.

CoralUI's aim is to provid everything your product needs to adopt the CloudUI visual style.

Text Styles

CUI provides basic text styles suitable for paragraph. When laying out your page, you might need to override the margins on the heading elements.

Headings

Use the <h1> through <h5> tags to add headings to sections on your page.

Page Components

Page components include header, rail, etc.

Header

The top navigation header that appears on every page. It supports a logo, navigation, and drawer (which can contain a badge or other information).

The HTML5 <header> tag reserved for the application header and should not be used elsewhere.

Home Activities Admin
5
<header>
  <div class="logo"><img src="images/theme/dms.png"></div>
  <nav>
    <a href="#" class="selected">Home</a>
    <a href="#">Activities</a>
    <a href="#">Admin</a>
  </nav>
  <div class="drawer">
    <div class="badge">5</div>
  </div>
</header>

Spinner

          
Use a spinner to indicate a process is in progress that will occur for an indefinite amount of time.

          
Applying the CSS class spinner to an element will make it inline-block.

<div class="spinner large"></div>

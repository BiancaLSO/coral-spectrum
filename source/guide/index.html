What is CoralUI?

CoralUI or CUI (pronounced "cooey") is an implementation of the CloudUI visual style from Adobe XD.

CoralUI's aim is to provide everything your product needs to adopt the CloudUI visual style.

Resolution Independence

CoralUI is resolution independent. By using elastic measurement units (em and rem) to define sizes, CoralUI can be scaled to the perfect size for any display, regardless of size or pixel density.

Browser Support

CoralUI supports all modern browsers, including IE 9+, Firefox (latest), Chrome (latest), Safari (latest).

Prevalent mobile devices are also be supported, including Android 2.1+ and iOS 5+.

Widget Philosophy

All CoralUI widgets are guaranteed to fulfill 7 promises:

  1. Widgets will be available as a jQuery plugin: $('#myModal').modal()
  2. Widgets will be available as a class: myModal = new CUI.Modal({ element: '#myModal' }).
  3. Widget APIs will always be fully documented here.
  4. Widgets will always be easily extendable with CUI.Modal.extend().
  5. Widgets will progressively enhance existing markup, or render a template using passed options if no markup is present.
  6. Widgets will take advantage of declarative markup using data-* attributes.
  7. Widgets will work on both desktop and mobile.

Page Components

Page components include header, rail, etc.

Header

The top navigation header that appears on every page. It supports a logo, navigation, and drawer (which can contain a badge or other information).

The HTML5 <header> tag reserved for the application header and should not be used elsewhere.

<!-- Created from Markup -->
<div id="modal_HTML" class="modal">
  <div class="modal-header">
    <button type="button" class="close" data-dismiss="modal">&times;</button>
    <h2>Modal from Markup</h2>
  </div>
  <div class="modal-body">
    <p>This modal was created from markup.</p>
  </div>
  <div class="modal-footer">
    <button data-dismiss="modal">Close</button>
    <button class="primary" data-dismiss="modal">Save</button>
  </div>
</div>

<!-- Created with JavaScript -->
<div id="modal_JS"></div>

<script type="text/javascript">
  $('#modal_JS').modal({
    heading: 'Modal from JS',
    content: '<p>This modal was created with JS.</p>',
    buttons: [
      {
        label: 'Close',
        click: 'close',
      },
      {
        label: 'Save',
        className: 'primary',
        click: function(evt) {
          console.log('Modal: This would usually trigger a save...');
          evt.dialog.hide();
        }
      }
    ],
    visible: false
  });
</script>
